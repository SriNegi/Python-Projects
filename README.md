# Python-Projects
# 1 Python Calculator

This is a simple command-line calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Exit option
- Uses match-case (Python 3.10+)
# 2 Number Guessing Game

This is a simple number guessing game built using Python.

## Features

- Random number generation between 1 and 50
- Tracks number of attempts
- Hints if guess is too high or too low
- Shows congratulations when guessed correctly

## How to Run
1. Make sure Python 3.10+ is installed.
2. Run the file:
   python calculator.py
