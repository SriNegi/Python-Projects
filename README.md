# Python-Projects
# 1 Python Calculator

This is a simple command-line calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Exit option
- Uses match-case (Python 3.10+)

## How to Run
1. Make sure Python 3.10+ is installed.
2. Run the file:
   python calculator.py
